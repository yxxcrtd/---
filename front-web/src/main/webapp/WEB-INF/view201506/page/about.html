关于财猫

财猫简介：

财猫网互联网证券交易平台由北京朋乐投资管理有限公司全资负责运营，注册资金1000万人民币。公司于2015年4月成立于北京中关村高科技产业园区上地信息技术基地，组建了一只由金融服务、风险管理及互联网运营等业界一流的专业人士组成的团队。

财猫业务：

财猫网致力于为用户提供一个涵盖全球主流投资品种的一站式的互联网证券投资交易平台。目前财猫网已经开通贵金属交易服务，即将开通外汇和美股交易服务，后续将为用户提供更多品类的证券投资服务。

财猫团队：

核心团队成员多毕业于清华大学、北京大学、复旦大学等国内顶级名校，曾就职于甲骨文、腾讯、中信证券、阿里巴巴、美团、恒生电子等国内外知名企业，是国内最专业的互联网金融团队之一。
