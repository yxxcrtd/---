1、邮币卡：是指具有投资价值的邮票、钱币、电话卡，已成为拓展人们集藏与投资渠道的新趋势。现在，邮币卡电子盘令交易者可以像投资股票一样投资买卖邮币卡藏品。

2、交易所：指在财猫网上展示的全部与财猫网合作的交易所，授权我们为其提供投资者代理开户及贵金属交易指令提交业务。

5、本公司：指北京朋乐投资管理有限公司。

四、服务内容

成为财猫网用户后，用户可以获得财猫网提供的以下服务：

1、财猫邮币卡：

1.1、财猫网与邮币卡交易所合作，用户在财猫网提交各个交易所要求的开户资料（包括但不限于手机号码、身份证件照片、银行卡号等信息），如果用户提交的材料通过了财猫网后台审核，我们将会将用户的开户信息提交到交易所完成开户；完成开户后，用户会获得相应的交易所的账户名和密码等信息，用户须妥善保管上述信息。

1.2、用户只要在财猫网上提交一次资料(资料符合交易所要求的前提下)，就可以完成我们合作的多家交易所开户流程，开启邮币卡交易。

1.3、财猫网将其接入的各大邮币卡交易所的行情信息，分时、K线等实时交易信息展示在财猫网上，用户可以随时查看。

2、财猫贵金属：

2.1、财猫网与国内多家贵金属交易所保持合作关系，用户按照财猫网贵金属业务开户流程和要求完成开户、选择交易所进行银商绑定并同意该交易所交易规则后，即可获得通过财猫网在该交易所进行贵金属交易的资格。

2.2、用户完成贵金属业务开户相关流程后，即可通过财猫网浏览实时交易行情和资讯、通过财猫网提交贵金属交易指令并完成贵金属交易。

2.3、财猫网提供上海黄金交易所、南方稀贵金属交易所等合作交易所的的行情信息，贵金属交易分时、K线等实时交易信息供用户查看参考。

2.4、财猫网精选经济要闻、实时快讯等与贵金属交易相关的经济资讯展示在财猫网上，用户可以随时查看参考。

3、用户有权在财猫网查看其个人账号下的信息，有权应用财猫网提供的功能进行操作。

4、用户有权按照我们发布的活动规则参与财猫网组织的网站活动。

5、财猫网为用户提供部分交易所的交易咨询、发布交易所公告、打新申购、新闻报道的行业信息等服务，交易所的数量将随着我们的合作不断拓展，随时更新在财猫网上。
