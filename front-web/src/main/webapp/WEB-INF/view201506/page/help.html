财猫网Q/A

Q1：什么叫股票融资？

A1:很多投资者具有良好的盈利能力和风险控制能力，但受制于自身资金量较小，其盈利能力和资金管理的能力无法得到充分发挥。而解决这一问题最直接的方法就是扩大操作资金。股票融资是为具备丰富交易经验及风险控制能力的股票投资者提供的放大其操作资金的业务。

Q2：股票融资和融资融券的区别？

A2:股票融资和券商提供的融资融券业务从本质上讲都是可以增加您的操盘资金，但两者又有着很大的差别，先从资本概念上来分析一下融资与融资融券。
        股票融资是在需要一定保证金的情况下提供放大资金比例操盘，并只收取出借资金利息的业务；融资融券则是指投资者向具有上海证券交易所或深圳证券交易所会员资格的证券公司提供担保物，借入资金买入本所上市证券或借入本所上市证券并卖出的行为。但是在融资融券实际操作中存在着诸多的限制，以下是增财易对两项业务的不同点分别作出的对比： 融资融券的限制（括号里的内容是与股票融资的对比）：
        第一：开户必须满18个月（融资公司提供的股票融资业务并无此限制）。
        第二：市值最低50万（融资公司提供的股票融资业务并无此限制，一般是1-5万起融）。
        第三：标地股共有300只A股可以操作，同样会被机构操作。（通常融资公司对于st、*st、创业板、新股上市当天等没有涨跌停限制的股票是要求不能交易的。但尽管如此，融资公司对股票的交易限制条件任然要比融资融券宽松的多）
        第四：融资额度只有25%（不同的融资公司对融资比例要求不同，有的是按照1:4，有的融资比例会更高）。
        第五：交易佣金高，通常在千分之一以上（而融资账户交易佣金通常在千分之一以下）
        第六：最长期限6个月必须交割（现在部分融资公司对合作时间有所要求，要求必须要做够3个月，也有部分融资公司没有这个限制，一个月起。还有的极少部分融资公司采取按天收费的形式）。
        第七：平仓线较高最大限度不允许低于175%（融资合作中平仓线是自有资金即保证金亏损超过60%左右是要平仓的）。
        第八：需要提交足额的担保物（融资公司并无此限制）

Q3：利息怎么计算？

A3:利息按月或按日计算，先支付利息后使用资金，如：3月1日融资，融资时支付第一个月利息，下月开始时支付第二个月的利息，以此类推。其中利息不包含印花税，过户费和佣金。印花税，过户费和佣金直接由券商和交易所收取。

Q4：亏损警告线是？

A4:账户资金达到融资金额*1.15即为亏损警告线。请您尽快补充风险保证金，以免被平仓。

Q5：亏损平仓线是？

A5:账户资金达到融资金额*1.1，即为亏损平仓线。当账户资金低于平仓线以下时，平台有权进行强制平仓。为避免平仓发生，请及时关注风险保证金是否充足。

Q6：资金使用期限是多长时间？

A6:月融：使用期为1个月，到期之前可以续约。
      日融：使用一天算一天

Q7：交易时间是什么时间？

A7:每天早上9:30-11:30 下午13：00-15:00

Q8：交易手续费是多少？

A8:手续费包含印花税、过户费和交易佣金，印花税和过户费按财政部和交易所规定收取，交易佣金0.08%（万8）由券商收取

Q9：盈利如何提取？

A9:用户申请提现，审核通过后就可以提现

Q10：是否需要进行股票开户，开户费是多少？

A10:您不需要去证券公司开户，统一采用我们在证券公司已经开好的股票账户，成功申请融资后会直接把现成的账户跟密码给您进行操盘的。

Q11：融资是否需要抵押或者担保？

A11:普通用户不需要额外的抵押或担保

Q12：电子合同是否有法律保证

A12:根据《合同法》和《电子签名法》的规定，当事人可以采用合同书、信件和数据电文（包括电报、电传、传真、电子数据交换和电子邮件）等形式订立合同，并通过以电子形式所含、所附用于识别签名人身份并表明签名人认可其中内容的数据电文等电子签名方式进行签署，当事人不能仅因合同采用电子签名、数据电文的形式就否定其法律效力。

Q13：我的资金安全吗

A13:您的每一笔保证金的流转在银行均有记录，财猫网与您签署的协议符合民间借贷法律特征，受法律保护。

Q14：融资合法吗？

A14:融资行为是符合国家法律规定的。融资属于民间借贷的一种形式，《合同法》和《最高人民法院关于人民法院审理借贷案件的若干意见》等都从法律层面上肯定了融资行为的合法性。
