操盘演示（例：投资本金1万，融资5万，总操盘资金6万，每月利息950元）

投资本金1万，融资5万，您的总操盘资金6万，每月利息950元，买入股票6万，并持有30天，股票上涨10％。

  • 第1天买入股票6万
  • 第2-29天继续持有
  • 第30天股票涨了10%，卖出股票，结束操盘

结算盈亏

股票盈利：60000元 x 10％ = 6000元

支付利息：950元

投资本金：10000元

您可以获得：6000元 - 950元 + 10000元 = 15050元

投资收益率：（6000元 - 950元）÷ 10000元 = 50.5％

融资盈利演示

投资本金1万，融资5万，您的总操盘资金6万，每月利息950元，买入股票6万，并持有30天，股票下跌2％。

  • 第1天买入股票6万
  • 第2-29天继续持有
  • 第30天股票下跌了2％，卖出股票，结束操盘

结算盈亏

股票亏损：60000元 x 2％ = 1200元

支付利息：950元

投资本金：10000元

您可以获得：10000元 - 950元 - 1200元 = 7850元

投资亏损率：（10000元 - 7850元）÷ 10000元 = 21.5％
