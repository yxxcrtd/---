活动规划

1.日融资用户前3天免息；月融资用户前3天免息。（每名用户只限一次免息）

2.返息时间：每日16:00前融资的用户，当日16:00-18:00返息到账户；每日16:00后融资的用户，次日16:00-18:00返息到账户。

3.活动期间月融资额度超过30万，送招财猫一个。

4.活动期间月融资额度超过100万，送50G银币一枚。

5.活动期间月融资额度超过200万，送Apple Watch一个。

6.活动期间月融资额度超过300万，送iPhone6一个。

7.活动奖品在活动结束后统一发放

8.活动时间：2015年5月12日-2015年5月22日

9.本活动最终解释权归财猫网所有。
